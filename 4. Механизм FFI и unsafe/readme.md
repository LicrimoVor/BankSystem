#  FFI и unsafe

Эта папка содержит примеры использования FFI (Foreign Function Interface) и unsafe кода в Rust.

## Проекты

### 1. static_links

Пример статической линковки C-библиотеки с Rust.

**Особенности:**

- Использует `bindgen` для генерации биндингов
- Статическая линковка C-кода через `cc` crate
- Автоматическая генерация Rust-оберток для C-функций

**Структура:**

- `src/mylib.c` - C-реализация
- `src/mylib.h` - C-заголовки
- `build.rs` - скрипт сборки с генерацией биндингов

### 2. dynamic_links

Пример динамической линковки C-библиотеки с Rust.

**Особенности:**

- Использует `libloading` для загрузки динамических библиотек
- Работа с функциями через указатели
- Обработка ошибок при загрузке библиотек

**Структура:**

- `src/mylib.c` - C-реализация
- `src/mylib.h` - C-заголовки
- `src/main.rs` - Rust код с динамической загрузкой

### 3. mylib

Пример создания Rust-библиотеки для использования из других языков.

**Особенности:**

- Экспорт функций с `#[no_mangle]` и `extern "C"`
- Поддержка нескольких типов сборки (dylib, staticlib)
- Примеры использования из C и Python

**Примеры использования:**

- `examples/test.c` - использование из C
- `examples/test.py` - использование из Python через ctypes

### 4. image_project

Комплексный пример обработки изображений с использованием плагинов.

**Особенности:**

- CLI-процессор изображений
- Поддержка FFI-плагинов
- Загрузка динамических библиотек
- Работа с изображениями через `image` crate

**Компоненты:**

- `processor/` - основной процессор
- `plugin_blur/` - плагин размытия
- `plugin_mirror/` - плагин отзеркаливания
- `py_processor/` - Python версия процессора
