# Реализация упрощённого Proof of Work

## Основные реализовыванные функции

- Структура данных для моделирования блокчейн
- Возможность майнить новые блоки с помощью алгоритма Proof of Work с фиксированной сложностью
- Реализация удобного REST API, через который возможно получать и добавлять блоки и транзакции.

## Начало работы

Необходимо чтобы Rust и Cargo были предустановленны.

```sh
cargo run
```

Приложение запустит веб сервер и будет слушать 3000 порт. Веб сервер имеер три ендпоинта: "/", "/blocks", "/txs"

| Method | URL     | Description                                |
| ------ | ------- | ------------------------------------------ |
| Get    | /       | Healthcheck                                |
| Get    | /blocks | Вывести все блоки блочейн                  |
| Post   | /blocks | Добавить новый блок                        |
| Get    | /txs    | Вывлести все транзакции находящиеся в пуле |
| Post   | /txs    | Добавить новую транзакцию в пул            |
